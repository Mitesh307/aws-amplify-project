# 🚀 AWS Amplify CI/CD Website Deployment

This project demonstrates a complete CI/CD deployment workflow using GitHub and AWS Amplify. The website is automatically built and deployed whenever new code is pushed to the GitHub repository.

⚡ Auto Deployment Workflow

Every time code is pushed using Git:

git add .
git commit -m "update"
git push

AWS Amplify automatically:

Detects code changes
Starts build process
Deploys updated version
Updates live website instantly
